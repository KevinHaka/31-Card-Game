# 31 Card Game

This project is a Python implementation of the classic 31 Card Game featuring a graphical user interface (GUI).

## About

This is the final project for the course *"Advanced programming in Python"* offered by Mathesis (mathesis.cup.gr). The project demonstrates the application of advanced Python programming concepts, including object-oriented programming and GUI development. I have implemented approximately 10% of the code within the `thirtyone_gui.py` file.

## How to Run

1. Clone the repository:
    ```sh
    git clone https://github.com/KevinHaka/31-Card-Game.git
    ```
2. Navigate to the project directory:
    ```sh
    cd 31-Card-Game
    ```
3. Run the application:
    ```sh
    python thirtyone_gui.py
    ```

## Usage

- Start the game.
- Interact with the graphical interface to draw, hold, and play your cards to reach 31.
